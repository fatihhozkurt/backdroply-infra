# Quick Guide

## TR

1. Landing sayfasinda kisa guide popup'ini tamamla.
2. Google ile giris yap, token bakiyeni sag ustte kontrol et.
3. `Studio`ya gec:
   - `Gorsel` veya `Video` sec
   - kalite sec (`Balanced` veya `Ultra`)
   - istersen brush panelinde `Keep/Erase` ciz
4. `Islemi Baslat` de.
5. Ciktiyi indir.
6. `My Media` aktifse son 10 cikti tutulur; 11. cikti geldiginde en eski kayit otomatik silinir.
7. Hesap silmek istersen Studio sag panelde `Hesabi Sil` butonunu kullan.

### Ornek Akis

- Siyah-beyaz dalgali arkaplanli video:
  - `Auto Ultra`
  - sorunlu saniyeyi frame olarak yukle
  - keep ile karakter+laptopu isaretle
  - tekrar process et

## EN

1. Complete the short onboarding popup on landing.
2. Sign in with Google and check token balance at top-right.
3. Open `Studio`:
   - choose `Image` or `Video`
   - pick quality (`Balanced` or `Ultra`)
   - optionally draw `Keep/Erase` masks
4. Run processing.
5. Download output.
6. If `My Media` is enabled, only the latest 10 outputs are kept; the oldest one is removed automatically when a new item arrives.
7. If needed, remove account from the Studio side panel via `Delete Account`.
